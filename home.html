Šta je Web stranica?

Web stranica je pojedinačni dokument ili jedna strana na bilo kojoj Web lokaciji, to jest Websajtu, koja se prikazuje u Web pretraživaču. Web stranica prikazuje sadržaj u vidu teksta, slika, videa, veza i drugih vrsta medija. Svaka Web stranica poseduje posebnu URL adresu koja se koristi kako bi se stranici pristupilo i prikazao sadržaj na njoj. Primer za URL Web stranice bi bio http://pin.ekof.bg.ac.rs/predavanja.html – pristupa se Web stranici predavanja.html koja se nalazi na Websajtu Poslovne informatike, pa stoga samo http://pin.ekof.bg.ac.rs/ predstavlja URL Websajta Poslovne informatike.
